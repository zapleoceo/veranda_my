# TR4: Telegram hall_name + Table Badge UI + Fountain + TR3 Redirect + Refactor (Design)

- Увеличить “радиус воды” у фонтана на 10% и поднять фонтан вверх на величину его радиуса.
- Перенести “номер стола” и “ёмкость” в левый верхний угол каждого стола в TR4 и сделать для них отдельную оранжевую подложку (цветом стола).
- Провести рефактор TR4: убрать дублирование, выделить ответственности (без сборщика).
- Сделать редирект со страницы `/tr3/` на `/tr4/`.
- Поднять версии ассетов и задеплоить.

## Неголсы
- Полная замена TR3 API или изменение бизнес-логики TR3.
- Введение сборщика/бандлера (webpack/vite) или переход на SPA-фреймворк.

## Изменения данных

### 1) `reservations` (таблица бронирований)
Добавить поля:
- `spot_id` int null
- `hall_id` int null

Зачем:
- для построения Telegram текста требуется `hall_name`, который определяется по `spot_id + hall_id`.

Источник значений:
- TR4 UI знает активный `hall_id` и `spot_id` (уже используется в запросах к Poster) и передаёт их в `submit_booking`.

### 2) Кэш hall_name
Добавить хранение списка залов из Poster `spots.getSpotTablesHalls`:
- минимум: через `system_meta` (ключ вида `poster_spot_halls_<spot_id>`) с TTL
- либо отдельная таблица (опционально, если meta не подходит)

Требования к кэшу:
- TTL по умолчанию 12 часов.
- При отсутствии/ошибке Poster: fallback на строку `hall_id=<id>`.

## Интеграция с Poster
- Использовать `spots.getSpotTablesHalls` для получения списка залов (id → name).
- Нормализовать `hall_name` как строку; хранить только то, что нужно для Telegram текста.

## Telegram тексты

### 1) Менеджер
- В `ReservationTelegram::buildManagerText()` заменить строку про стол на формат:
  - `Стол: <hall_name> · <table_id>`

### 2) Гость (Telegram)
- В `tr4/api_booking.php` (формирование `$userText`) заменить строку про стол на тот же формат:
  - `Стол: <hall_name> · <table_id>`

Fallback:
- если `hall_name` неизвестен, использовать `hall_id=<id>`.
- если `poster_table_id` отсутствует, использовать `table_num` как fallback для совместимости.

## UI TR4 (столики)

- В badge:

  - капасити (👥 N)

### Стили

- у badge собственная подложка:
  - оранжевый фон как у стола (тот же gradient/цветовые переменные)
  - белый текст
  - компактный padding и скругление

## Фонтан

### Радиус воды +10%
- В SVG фонтана увеличить радиус соответствующего “водяного” круга на 10%.

### Поднять фонтан вверх на радиус
